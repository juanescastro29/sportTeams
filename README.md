# Título del Proyecto

Sport Teams

## Comenzando 🚀
Se ejecuta en el puerto 3000
http://localhost:3000/

### Pre-requisitos 📋

Utilizando node js
Express
Motor de vistas ejs
Nodemon
Bootstrap

### Instalación 🔧
Como dependencias se utilizo express
npm i express 

Nodemon para el reinicio del servidor automaticamente al guardar los cambios:
npm i nodemon 

Bootstrap v5.3
npm i bootstrap

Bootstrap Icons

npm i bootstrap-icons
