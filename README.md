# Título del Proyecto

Parcial desarrollo web

## Comenzando 🚀
Se ejecuta en el puerto 3000
http://localhost:3000/

### Pre-requisitos 📋

Utilizando node js
Express
Gestor ejs
Nodemon
Bootstrap

### Instalación 🔧
Como dependencias se utilizo express
npm i express --save

Nodemon para el reinicio del servidor automaticamente al guardar los cambios:
npm i nodemon --save
